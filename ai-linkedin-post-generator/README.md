# AI-powered LinkedIn Post Generator

## Spec
- **Goal**: Provide a simple web app where users input a topic, target audience, and key benefits, and receive a ready‑to‑post LinkedIn post generated by an LLM.
- **Core Features**:
  - Input form (topic, audience, key points)
  - Generate post via OpenAI API (gpt‑4o‑mini or similar)
  - Display generated post with copy‑to‑clipboard button
  - Basic usage analytics (posts generated per day)
  - Pricing page with free tier (3 posts/week) and paid tier (unlimited)
- **Tech Stack**:
  - **Frontend**: Next.js 13 (React 18), optional Tailwind CSS for styling
  - **Backend**: API route in Next.js calling OpenAI
  - **Deployment**: Vercel (serverless functions)
  - **Auth**: Optional Vercel/NextAuth for paid tier (out of scope for MVP)
- **MVP**: Minimal UI, generate button, display result, no auth.

## Development
```bash
npm install
npm run dev
```

Deploy with:
```bash
vercel --prod
```
